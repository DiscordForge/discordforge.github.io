People like modding, don't they?

DiscordForge brings that possibility to the ever-expanding Discord platform.

About

DiscordForge is a toolkit that allows you to create and install your own plugins into the Discord client, whether it be Stable, PTB, Canary, and (if you got it working) Development. We try to make the process simple by using the language that Discord is using: JavaScript.

How does it work?

When DiscordForge installs itself into your client, it modifies your client in a way that allows it to update DiscordForge without having to put it back in your client every single time you update it. Just update DiscordForge and restart Discord. Simples!

Installation

The overall installation of the client is actually rather simple.

  1. get git (here)
  2. get node (here)
  3. mkdir discord-forge
  4. git clone https://github.com/DiscordForge/DiscordForge discord-forge
  5. node setup

And you're done! If you want to install it into your client, type into a terminal discordforge inject
